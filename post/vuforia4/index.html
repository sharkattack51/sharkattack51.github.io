新機能

  • iOS 64bit サポート
  • オブジェクトトラッキング
    • 事前の３Dトラッキング情報を利用する
      • AndroidのScannerアプリによって特徴点データ生成
        • トラッキング基準としてA4サイズまたはハガキサイズのマーカーを利用する
        • 撮影時、上記マーカーが撮影アングルに入るような物理的な卓上サイズでなければスキャンできない
        • 特徴的な形状または、表面模様を検出する。検出ルールはおそらく画像マーカーと同様。
        • 平坦なライティング環境が良い
        • グレーの机が良い
    • たぶん下記は認識アウト
      • スキャン時と違いすぎる照明環境
      • 変形するもの
      • 反射するもの
      • 見上げて認識させようとする場合(スキャン時は見下げの状態なので)
      • 特徴的な凸凹が無い形状
      • パターン的な表面
    • スキャン方法
      1. Androidのスキャナーアプリで点群データを計測
        • 現在はDeveloperPotalから.apkをダウンロードし、手動でインストール
      2. .odデータを書き出す
        • 端末からメールなどを利用して送信
      3. DeveloperPortalでターゲットデータベースに登録
        • アプリケーションのライセンスに紐付け
      4. Unityパッケージデータをダウンロード
      5. アプリ組み込み
