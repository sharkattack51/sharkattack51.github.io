logrotate
$ yum install logrotate

  • cronによりdailyでスケジューリングされている

    $ cat /etc/cron.daily/logrotate

基本設定ファイル

$ cat /etc/logrotate.conf

サービスごとの設定ファイル

$ cat /etc/logrotate.d

ローテション確認

$ logrotate -dv /etc/logrotate.d/httpd
