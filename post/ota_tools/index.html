ota-tools

OTAインストールの準備および、証明書の入れ替えを行うmac用のコマンドラインツール。 プロビジョニングの入れ替えおよび、配布用に必要なplist、htmlを自動で生成する。

https://github.com/RichardBronosky/ota-tools

サインニング時

  • サインニングの入れ替えを行う

    $ cd ota-tools
$ ./ipa_sign XXXX.ipa XXX.mobileprovision "証明書登録名"

OTA配布時

  • 配布用plistとhtmlを生成

    $ cd ota-tools
$ ./ipa_ota XXXX.ipa https://ipa配置のURL/ > index.html

  • インストール用のリンクとサーバー証明書を配布

    • 構成プロファイルをメール等で配布し、iPhone構成ユーティリティで端末にインストールしてもらう
