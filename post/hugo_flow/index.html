Hugoのワークフロー

このサイトはHugoを使っている。そして頻繁に更新しない。 ので、新規ポストのたびに使い方調べる羽目になってつらい。ので、使い方思い出す用のメモ。

ポストの新規作成

$ hugo new post/post_name.md

ローカルサーバーで確認

$ hugo server --buildDrafts --watch

$ hugo server -D -w

パブリッシュ

$ hugo

デプロイ

$ git push origin master
